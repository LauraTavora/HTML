O que é HTML?

A sigla HTML significa Hyper Text Markup Language. É uma linguagem utilizada para produzir páginas na internet. Com essa linguagem,podemos criar páginas que contenham informação nos mais variados formatos: texto, som, imagens e animações.

Quais as funções do HTML?

A função do HTML (Hypertext Markup Language) na programação web sofreu alterações ao longo dos anos. E hoje essa linguagem deve ser utilizada unicamente para estruturar o conteúdo das páginas. Ou seja, não cabe a ela definir características visuais ou comportamentos. Isso deve ser feito usando CSS e JavaScript.
